Project information
===================
The main project website for FlexDi may be found at:

  https://github.com/csf-dev/FlexDi

Please refer here for documentation, upgrade news and to
file bug reports or feature requests.

FlexDi is based upon (and retains some source code from)
another project named BoDi:

  https://github.com/gasparnagy/BoDi
